# ICP Filing Checklist

Guide for ICP beiang (ICP备案) — mandatory domain registration for China-hosted websites.

---

## Quick Overview

| Aspect | Details |
|--------|---------|
| **Who needs it?** | ALL China-hosted websites/domains |
| **Authority** | Ministry of Industry and Information Technology (MIIT) |
| **Portal** | https://icp.beian.miit.gov.cn |
| **Cost** | FREE (no government fee) |
| **Timeline** | 7-20 working days |

---

## Required Documents Checklist

| Document | Required? | Source |
|----------|-----------|--------|
| Business License (营业执照) | ✅ LLC | Company registration |
| Domain Certificate (域名证书) | ✅ Yes | Domain registrar |
| Hosting Certificate (托管证书) | ✅ Yes | Hosting provider |
| Responsible Person ID (负责人身份证) | ✅ Yes | Chinese ID card |
| Emergency Contact (应急联系人) | ✅ Yes | Phone number |
| Website Screenshots (网站截图) | ✅ Yes | 5 screenshots |

**Status: [ ⬜ ⬜ ⬜ ⬜ ⬜ ⬜ ]**

## Filing Process

### Phase 1: Preparation (Days 1-2)

- [ ] Register domain
- [ ] Get domain certificate from registrar
- [ ] Set up China-based hosting
- [ ] Get hosting certificate
- [ ] Prepare 5 website screenshots:
  - [ ] Homepage
  - [ ] Features page
  - [ ] Privacy Policy
  - [ ] Terms of Service
  - [ ] Contact page

### Phase 2: Online Application (Day 3)

- [ ] Visit https://icp.beian.miit.gov.cn
- [ ] Register account
- [ ] Verify phone number
- [ ] Fill application:
  - [ ] Entity type: Enterprise (企业) or Individual (个人)
  - [ ] Entity name: As registered
  - [ ] Website type: Interactive (交互式)
  - [ ] Content: Accurate description
- [ ] Upload all documents
- [ ] Submit for review

### Phase 3: Review & Approval (Days 4-18)

| Stage | Duration | What Happens |
|-------|----------|-------------|
| Initial review | 1-3 days | Basic compliance check |
| Content review | 5-15 days | Website examination |
| Verification | 1-3 days | Phone verification |
| Final approval | 1-2 days | ICP number issued |

### Phase 4: Post-Approval (Day 19-20)

- [ ] Receive ICP number
- [ ] Display on website footer
- [ ] Link to https://beian.miit.gov.cn/

## Common Rejection Reasons & Solutions

| Rejection | Solution |
|-----------|----------|
| Domain not matching business | Ensure domain relates to business |
| Insufficient content | Add more pages (FAQ, About, Contact) |
| Hosting not in China | Use China-based hosting |
| Prohibited content | Remove political/religious/sensitive content |
| Screenshots unclear | Take clear, high-resolution screenshots |

---

## LLC vs Individual Filing

| Type | Eligibility | Pros | Cons |
|------|------------|------|------|
| **LLC (企业)** | Registered company | ✅ Can file directly | Requires business license |
| **Individual (个人)** | Chinese citizen | ✅ Simpler process | ❌ Personal liability |

**Recommendation:** File as LLC (企业) for commercial platforms.

---

## WeChat Mini Programs

| Requirement | Status |
|-------------|--------|
| ICP required? | ✅ YES, if external website domain |
| WeChat subdomain? | ✅ WeChat handles ICP collectively |
| Custom domain? | ✅ Requires separate ICP filing |

**Recommendation:** Start with WeChat subdomain, file custom domain later.

## Cost Breakdown

| Item | Cost |
|------|------|
| ICP filing | FREE |
| Domain registration | ¥30-100/year |
| China hosting | ¥500-2,000/year |
| SSL certificate | FREE (Let's Encrypt) or paid |

**Note:** Some providers charge "service fees" for filing assistance.
