# ICP Filing Required Documents

**Source:** https://icp.beian.miit.gov.cn/
**Last Accessed:** 2025-02-28
**Language:** English

## Required Documents Checklist

---

## For Enterprises (LLC/Company)

### 1. Business License (营业执照) ✅
- **Format:** Color scan or photo
- **Requirements:**
  - Must be current (not expired)
  - Shows company name, registration number
  - Legal representative name
  - Business scope
- **Source:** Company registration authority

### 2. Domain Certificate (域名证书) ✅
- **Format:** Official certificate from domain registrar
- **Requirements:**
  - Domain owner matches business license name
  - Not expired
  - Shows domain registration date
- **Source:** Domain registrar (Aliyun, Tencent, etc.)

### 3. Hosting Certificate (托管证书) ✅
- **Format:** Official certificate from hosting provider
- **Requirements:**
  - China-based hosting
  - Service period valid
  - Company name matches business license
- **Source:** Hosting provider

### 4. Responsible Person ID (负责人身份证) ✅
- **Format:** Color scan of ID card (both sides)
- **Requirements:**
  - Must be Chinese citizen ID
  - Current, not expired
  - Clear photo and text
  - Can be legal representative or authorized person
- **Source:** Individual's ID card

### 5. Emergency Contact Information ✅
- **Format:** Phone number and verification
- **Requirements:**
  - Working phone number
  - Can be verified via SMS
- **Source:** Company records

### 6. Website Screenshots (网站截图) ✅
- **Format:** 5 screenshots showing:
  1. Homepage
  2. Features/Services page
  3. Privacy Policy page
  4. Terms of Service page
  5. Contact/About page
- **Requirements:**
  - High resolution
  - Shows website URL
  - Pages are accessible (not 404)
  - Content matches filing description

---

## For Individuals (个人)

### 1. ID Card (身份证) ✅
- **Format:** Color scan (both sides)
- **Requirements:**
  - Chinese citizen ID
  - Current, not expired
  - Clear photo and text

### 2. Domain Certificate ✅
- Same as enterprise requirements
- Domain owner matches individual name

### 3. Hosting Certificate ✅
- Same as enterprise requirements
- Shows individual name

### 4. Emergency Contact ✅
- Same as enterprise requirements

### 5. Website Screenshots ✅
- Same as enterprise requirements

---

## Document Format Requirements

| Requirement | Specification |
|-------------|----------------|
| File format | JPG, PNG, PDF |
| File size | < 5MB per file |
| Resolution | Clear, readable text |
| Color | Color scan/photo (not black & white) |
| Authenticity | No alterations or modifications |

---

## Tips for Successful Filing

1. **Name Consistency:** All documents must use consistent names
   - Domain owner = Business license holder
   - Hosting account holder = Business license holder

2. **China Hosting:** Must use China-based hosting
   - Aliyun, Tencent Cloud, AWS China, etc.

3. **Content Quality:** Screenshots must show real content
   - Not "under construction"
   - Has actual pages (not blank)

4. **Phone Verification:** Ensure phone number can receive SMS
   - Emergency contact will be called for verification
