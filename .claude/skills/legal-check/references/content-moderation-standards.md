# Content Moderation Standards

**Source:** Compiled from CAC guidelines, WeChat policies, and industry standards
**Last Accessed:** 2025-02-28
**Language:** English

## Content Moderation Standards for Chinese Digital Platforms

---

## Regulatory Framework

### Primary Regulations
- **Cybersecurity Law (2016)** - General content obligations
- **E-Commerce Law (2019)** - Platform liability
- **CAC Content Guidelines** - Detailed prohibited categories
- **Platform-Specific Rules** - WeChat, Douyin, Bilibili policies

---

## Core Moderation Categories

### 1. Political Content (Prohibited)
| Type | Description |
|------|-------------|
| State secrets | Any classified government information |
| Subversion | Content opposing constitutional principles |
| National unity | Content damaging ethnic unity |
- Territory integrity | Content advocating secession |
- Rumors | False information disrupting social order |

### 2. Religious Content (Restricted)
| Type | Description |
|------|-------------|
| Proselytizing | Active recruitment/conversion |
| Extremism | Radical religious content |
| Sectarian | Content promoting division |
- Foreign missionary | Unauthorized foreign religious activity |

**Allowed:** Cultural discussion, historical context
**Not Allowed:** Conversion attempts, religious recruitment

### 3. Adult Content (Prohibited)
| Type | Description |
|------|-------------|
| Pornography | Explicit sexual content |
| Obscenity | Indecent or vulgar content |
- Sexual services | Solicitation or advertising |

### 4. Violence & Harm (Prohibited)
| Type | Description |
|------|-------------|
| Graphic violence | Bloody/gory content |
| Self-harm | Suicide, self-instruction |
- Dangerous activities | Instructions for harmful acts |
- Terrorism | Extremist content |

### 5. Fraud & Deception (Prohibited)
| Type | Description |
|------|-------------|
| Scams | Fraudulent schemes |
| False advertising | Misleading commercial content |
- Fake products | Counterfeit goods |
- Impersonation | Pretending to be official/other |

### 6. Social Harm (Prohibited)
| Type | Description |
|------|-------------|
| Harassment | Targeted abuse |
| Discrimination | Hate speech based on identity |
- Cyberbullying | Coordinated harassment |
| Doxxing | Revealing private information |

## Industry Standards

### Review Timing
| Platform | Typical Review | Human Review |
|----------|---------------|--------------|
| WeChat Mini Program | 1-7 days | For borderline content |
| Douyin | Hours to days | For viral content |
| Bilibili | Hours to days | For UPV content |

### Appeal Process
1. Content removed → Notification sent
2. User appeals within 7 days
3. Platform re-reviews (1-3 days)
4. Decision: uphold or restore

## Moderation Best Practices

### Pre-Publication Checklist
- [ ] No political content
- [ ] No religious proselytizing
- [ ] No sexual/violent content
- [ ] No misleading claims
- [ ] No hate speech
- [ ] No copyrighted content (without permission)

### Language to Avoid
| Category | Avoid |
|----------|-------|
| Political | "Democracy," "rights defense," political leaders |
| Religious | "Salvation," "conversion," religious recruitment |
| Social | "Protest," "petition," "rights" (in advocacy context) |

### Safe Alternatives
| Instead of | Use |
|------------|-----|
| Political activism | Community building |
| Religious content | Cultural content |
| "Rights" language | "Customer service," "Member benefits" |

## Platform-Specific Standards

### WeChat Mini Programs
- **Stricter** on political/religious content
- Focus on service value, not content creation
- User-generated content requires active moderation

### Douyin (TikTok China)
- Visual content focus
- Strong filters for violence/adult content
- Trending topics reviewed

### Bilibili
- Educational context more tolerated
- User community moderation
- Danmu (bullet comments) filtered

## Compliance Requirements

### Platform Obligations
1. **Content review system** - Automated + human
2. **Report mechanism** - User reporting tools
3. **Response timeline** - 24-48 hours for reports
4. **Record keeping** - 6+ months of moderation logs
5. **Transparency report** - Annual disclosure (large platforms)

### Penalties for Non-Compliance
| Violation | Penalty |
|-----------|---------|
| Failure to moderate | Platform shutdown |
| Insufficient moderation | Suspension + fine |
| Delayed response | Administrative penalty |
